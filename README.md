# ARJSProject

ARJSProject adalah aplikasi berbasis web yang menggunakan teknologi Augmented Reality (AR) untuk menampilkan model 3D Mikrotik dengan menggunakan marker. Aplikasi ini dibangun dengan menggunakan **React.js**, **A-Frame**, dan **AR.js**.

---

## Fitur
- Menampilkan model 3D Mikrotik secara interaktif menggunakan marker.
- Kompatibel dengan kamera laptop atau perangkat mobile.
- Mudah di-deploy dan dapat diakses secara online.

---

## Teknologi yang Digunakan
- **React.js**: Framework JavaScript untuk membangun antarmuka pengguna.
- **A-Frame**: Framework untuk membuat pengalaman WebVR dan WebAR.
- **AR.js**: Library Augmented Reality yang ringan dan cepat untuk aplikasi berbasis web.
